# Nate's Personal Portfolio Project

#### Video Demo:  https://www.youtube.com/watch?v=DIdnb8f06gg
#### Description: 
This personal portfolio website is designed to showcase my completed projects while providing an interactive space for users to learn more about me and my work. It serves as both a professional presentation tool and a hands-on demonstration of my full-stack development skills. The application is built using a combination of technologies including Python, Flask, SQL, Jinja, JavaScript, HTML, CSS, Node.js, React, and Bootstrap, allowing me to integrate both backend and frontend functionality into a cohesive experience. The website consists of a public-facing interface and a secure administrative dashboard. On the public side, users can browse through my projects, view detailed descriptions, and access links to my GitHub repositories. It also includes sections about me, along with links to my social profiles, making it easy for visitors to connect and explore my work further. The design emphasizes a modern and sleek aesthetic, with smooth navigation and a seamless scrolling experience across all sections of the homepage. To enhance user engagement, the frontend incorporates dynamic features such as animated tiles and menus, light and dark mode toggling, and an interactive background that responds to mouse movement. These elements bring life to the interface and create a more immersive browsing experience. On the backend, a full CRUD admin dashboard allows me to efficiently manage project entries. The SQL database stores project data, including cover images and GitHub links, as well as secure admin login credentials. Accessibility is also a priority, with ARIA labels implemented throughout the site to ensure usability for all users. Overall, this project is intended to feel like a polished, real-world product while demonstrating strong foundational development skills.
